  1. 3레벨 이상 차이나는 이미지 (최우선)
  2. 과대평가된 이미지 (실제보다 높게 평가)
  3. 정면각도에서 오분류된 이미지
  4. 상단각도에서 오분류된 이미지

재라벨링 기준:
  1. 전문가 2명 이상의 합의
  2. 노우드 분류 기준표 재참조
  3. 유사한 케이스들과의 일관성 확인
  4. 이미지 품질이 분석에 부적합한 경우 제외 고려

데이터셋 개선 제안:
  1. 각도별 균등한 분포 확보
  2. 경계 케이스(레벨 간 애매한 경우) 추가 수집
  3. 이미지 품질 표준화
  4. 전문가 검증 단계 추가
